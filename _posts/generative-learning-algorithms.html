判别学习算法：直接求后验概率p(y|x)，例如logistic回归

p(y∣x;θ)=(hθ(x))y(1−hθ(x))1−yp(y|x;\theta)=(h_{\theta}(x))^{y}(1-h_{\theta}(x))^{1-y}p(y∣x;θ)=(hθ​(x))y(1−hθ​(x))1−y

生成学习算法：先求p(x|y)和p(y)，再根据贝叶斯公式得到p(y|x)

p(y∣x)=p(x∣y)p(y)p(x)p(y|x)=\frac{p(x|y)p(y)}{p(x)}p(y∣x)=p(x)p(x∣y)p(y)​

其中，p(x)可以这样计算：

p(x)=p(x∣y=1)p(y=1)+p(x∣y=0)p(y=0)p(x)=p(x|y=1)p(y=1)+p(x|y=0)p(y=0)p(x)=p(x∣y=1)p(y=1)+p(x∣y=0)p(y=0)

那么，对于一个新的样本进行分类，即是求使得下式最大时的y：

argmax(p(y∣x))=argmaxp(x∣y)p(y)p(x)=argmaxp(x∣y)p(y)arg max(p(y|x))=arg max\frac{p(x|y)p(y)}{p(x)}=arg maxp(x|y)p(y)argmax(p(y∣x))=argmaxp(x)p(x∣y)p(y)​=argmaxp(x∣y)p(y)

多元高斯分布：x是一个n维的随机向量，对于一个分类问题，当y的取值固定时，对应的x都是服从多元高斯分布的，即x~N(u,∑)。多元高斯分布的概率密度函数表示为(对某个y)：

p(x;u,∑)=1(2π)n/2∣∑∣1/2exp(−1/2(x−u)T(∑)−1(x−u))p(x;u,\sum)=\frac{1}{(2\pi)^{n/2}|\sum|^{1/2}}exp(-1/2(x-u)^{T}(\sum)^{-1}(x-u))p(x;u,∑)=(2π)n/2∣∑∣1/21​exp(−1/2(x−u)T(∑)−1(x−u))

式中的u即当y固定时的样本均值，∑是样本的协方差矩阵(所有y共享)。当x是连续随机变量时：

u=E(X)=∫xxp(x;u,∑)dxu=E(X)=\int_{x}xp(x;u,\sum)dxu=E(X)=∫x​xp(x;u,∑)dx

而协方差矩阵定义为：

∑=∑i=1m(x−u)(x−u)T\sum=\sum_{i=1}^{m}(x-u)(x-u)^{T}∑=i=1∑m​(x−u)(x−u)T

高斯判别分析：当样本满足多元高斯分布时，可采用高斯判别分析进行分类。以二分类为例。

y∼Bernoulli(ϕ)y\sim Bernoulli(\phi )y∼Bernoulli(ϕ)p(x∣y=0)=1(2π)n/2∣∑∣1/2exp(−1/2(x−u0)T(∑)−1(x−u0))p(x|y=0)=\frac{1}{(2\pi)^{n/2}|\sum|^{1/2}}exp(-1/2(x-u_{0})^{T}(\sum)^{-1}(x-u_{0}))p(x∣y=0)=(2π)n/2∣∑∣1/21​exp(−1/2(x−u0​)T(∑)−1(x−u0​))p(x∣y=1)=1(2π)n/2∣∑∣1/2exp(−1/2(x−u1)T(∑)−1(x−u1))p(x|y=1)=\frac{1}{(2\pi)^{n/2}|\sum|^{1/2}}exp(-1/2(x-u_{1})^{T}(\sum)^{-1}(x-u_{1}))p(x∣y=1)=(2π)n/2∣∑∣1/21​exp(−1/2(x−u1​)T(∑)−1(x−u1​))

现在对Φ，u0，u1，Σ进行参数估计，对数联合似然函数可表示为：

l(ϕ,u0,u1,∑)=log∏i=1mp(x(i),y(i);u0,u1,∑)l(\phi,u_{0},u_{1},\sum)=log\prod_{i=1}^{m}p(x^{(i)},y^{(i)};u_{0},u_{1},\sum)l(ϕ,u0​,u1​,∑)=logi=1∏m​p(x(i),y(i);u0​,u1​,∑)=log∏i=1mp(x(i)∣y(i);u0,u1,∑)p(y(i);ϕ)=log\prod_{i=1}^{m}p(x^{(i)}|y^{(i)};u_{0},u_{1},\sum)p(y^{(i)};\phi)=logi=1∏m​p(x(i)∣y(i);u0​,u1​,∑)p(y(i);ϕ)

当上式取得最大值时，得到估计的参数为：

ϕ=1m∑i=1m1{y(i)=1}\phi=\frac{1}{m}\sum_{i=1}^{m}1\{y^{(i)}=1\}ϕ=m1​i=1∑m​1{y(i)=1}u0=∑i=1m1{y(i)=0}xi∑i=1m1{y(i)=0}u_{0}=\frac{\sum_{i=1}^{m}1\{y^{(i)}=0\}x^{i}}{\sum_{i=1}^{m}1\{y^{(i)}=0\}}u0​=∑i=1m​1{y(i)=0}∑i=1m​1{y(i)=0}xi​u1=∑i=1m1{y(i)=1}xi∑i=1m1{y(i)=1}u_{1}=\frac{\sum_{i=1}^{m}1\{y^{(i)}=1\}x^{i}}{\sum_{i=1}^{m}1\{y^{(i)}=1\}}u1​=∑i=1m​1{y(i)=1}∑i=1m​1{y(i)=1}xi​∑=1/m∑i=1m(x−uy(i))(x−uy(i))T\sum=1/m\sum_{i=1}^{m}(x-u_{y^{(i)}})(x-u_{y^{(i)}})^{T}∑=1/mi=1∑m​(x−uy(i)​)(x−uy(i)​)T

当p(x|y)满足多元高斯分布时，p{y|x}可以表示为sigmoid函数，反之不成立。也就是说高斯判别分析比logistic回归有更强的假设，当样本满足多元高斯分布时效果比后者更好。

另外，如果样本服从指数分布族，那么都可以用logistic回归来进行分类。

朴素贝叶斯

以垃圾邮件分类为例，词典的大小为n，邮件中的单词是否在词典出现作为特征向量的每一维，即输入是n维的向量。

如果直接对p(x|y)建模，x有2^n种可能，x满足多项分布，当n很大时直接计算的代价很大。

朴素贝叶斯基于一个很强的假设，即当y一定时，x的每一个分量是条件独立的：

p(xi∣y)=p(xi∣y,xj)p(x_{i}|y)=p(x_{i}|y,x_{j})p(xi​∣y)=p(xi​∣y,xj​)

于是有：

p(x∣y)=p(x1,x2...,xn∣y)=p(x1∣y)p(x2∣y,x1)...p(xn∣y,x1...,xn−1)p(x|y)=p(x_{1},x_{2}...,x_{n}|y)=p(x_{1}|y)p(x_{2}|y,x_{1})...p(x_{n}|y,x_{1}...,x_{n-1})p(x∣y)=p(x1​,x2​...,xn​∣y)=p(x1​∣y)p(x2​∣y,x1​)...p(xn​∣y,x1​...,xn−1​)=p(x1∣y)p(x2∣y)...p(xn∣y)=p(x_{1}|y)p(x_{2}|y)...p(x_{n}|y)=p(x1​∣y)p(x2​∣y)...p(xn​∣y)=∏i=1np(xi∣y)=\prod_{i=1}^{n}p(x_{i}|y)=i=1∏n​p(xi​∣y)

显然，y、xi|y都是服从伯努利分布：

y∼Bernoulli(ϕy)y\sim Bernoulli(\phi_{y})y∼Bernoulli(ϕy​)xi∣y∼Bernoulli(ϕi∣y)x_{i}|y\sim Bernoulli(\phi_{i|y})xi​∣y∼Bernoulli(ϕi∣y​)

定义联合似然函数：

L(ϕy,ϕi∣y=0,ϕi∣y=1)=∏i=1mp(x(i),y(i))L(\phi_{y},\phi_{i|y=0},\phi_{i|y=1})=\prod_{i=1}^{m}p(x^{(i)},y^{(i)})L(ϕy​,ϕi∣y=0​,ϕi∣y=1​)=i=1∏m​p(x(i),y(i))

最大化上式得到参数估计：

ϕy=∑i=1m1{y(i)=1}m\phi_{y}=\frac{\sum_{i=1}^{m}1\{y^{(i)}=1\}}{m}ϕy​=m∑i=1m​1{y(i)=1}​ϕj∣y=1=∑i=1m1{xj(i)=1&y(i)=1}∑i=1m1{y(i)=1}\phi_{j|y=1}=\frac{\sum_{i=1}^{m}1\{x^{(i)}_{j}=1 \& y^{(i)}=1 \}}{\sum_{i=1}^{m}1\{y^{(i)}=1\}}ϕj∣y=1​=∑i=1m​1{y(i)=1}∑i=1m​1{xj(i)​=1&y(i)=1}​ϕj∣y=0=∑i=1m1{xj(i)=1&y(i)=0}∑i=1m1{y(i)=0}\phi_{j|y=0}=\frac{\sum_{i=1}^{m}1\{x^{(i)}_{j}=1 \& y^{(i)}=0 \}}{\sum_{i=1}^{m}1\{y^{(i)}=0\}}ϕj∣y=0​=∑i=1m​1{y(i)=0}∑i=1m​1{xj(i)​=1&y(i)=0}​

对于给定的样本，预测其是垃圾邮件的概率：

p(y=1∣x)=p(x∣y=1)p(y=1)p(x)p(y=1|x)=\frac{p(x|y=1)p(y=1)}{p(x)}p(y=1∣x)=p(x)p(x∣y=1)p(y=1)​=(∏i=1np(xi∣y=1))p(y=1)p(x∣y=1)p(y=1)+p(x∣y=0)p(y=0)=\frac{(\prod_{i=1}^{n}p(x_{i}|y=1))p(y=1)}{p(x|y=1)p(y=1)+p(x|y=0)p(y=0)}=p(x∣y=1)p(y=1)+p(x∣y=0)p(y=0)(∏i=1n​p(xi​∣y=1))p(y=1)​=(∏i=1np(xi∣y=1))p(y=1)(∏i=1np(xi∣y=1))p(y=1)+(∏i=1np(xi∣y=0))p(y=0)=\frac{(\prod_{i=1}^{n}p(x_{i}|y=1))p(y=1)}{(\prod_{i=1}^{n}p(x_{i}|y=1))p(y=1)+(\prod_{i=1}^{n}p(x_{i}|y=0))p(y=0)}=(∏i=1n​p(xi​∣y=1))p(y=1)+(∏i=1n​p(xi​∣y=0))p(y=0)(∏i=1n​p(xi​∣y=1))p(y=1)​

其中，p(xi|y)计算公式如下：

p(xi∣y)=p(i∣y)xi+(1−p(i∣y))(1−xi)p(x_{i}|y)=p(i|y)x_{i}+(1-p(i|y))(1-x_{i})p(xi​∣y)=p(i∣y)xi​+(1−p(i∣y))(1−xi​)

以下为使用UCI的垃圾短信数据集SMS Spam Collection Data Set进行分类的实现代码(链接)：

  1. 加载数据集
# 1. load samples.
X, y = load_samples()

# choose 70% of X as train data, others as test data.
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3)
m, n = X_train.shape
y_train = np.array(y_train).reshape(m, 1)
  1. 训练模型
# 2. parameter estimate using Laplace smoothing.
phi_y = sum(y_train) / float(m)
phi_1 = (sum(X_train * y_train) + 1) / (float(sum(y_train)) + 2)
phi_0 = (sum(X_train * (y_train ^ 1)) + 1) / ((m - float(sum(y_train))) + 2)
  1. 测试阶段，由于p(x)与分类无关，根据贝叶斯公式计算p(y|x)时忽略分母：
# 3. test
m, n = X_test.shape
y_test = np.array(y_test).reshape(m, 1)

p_1 = (X_test * phi_1 + (1 - phi_1) * (1 - X_test)).prod(axis = 1) * phi_y
p_0 = (X_test * phi_0 + (1 - phi_0) * (1 - X_test)).prod(axis = 1) * (1 - phi_y)
h = (p_1 >= p_0).reshape(m, 1)
print('accuracy rate: %.2f%%' % (np.sum(h == y_test) * 100 / float(m)))
  1. 与scikit的实现进行对比，结果表明二者的准确度相同，都在95%以上：
# 4. compare with scikit
bnb = BernoulliNB()
h = bnb.fit(X_train, y_train.ravel()).predict(X_test)
print('accuracy rate of scikit: %.2f%%' % (bnb.fit(X_train, y_train.ravel()).score(X_test, y_test.ravel()) * 100))
